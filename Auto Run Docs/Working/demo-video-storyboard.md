## Production Notes

### Screen Recording Requirements

1. **Clean Environment**
   - Minimal desktop background
   - Only necessary applications visible
   - Disable desktop notifications during recording

2. **Recommended Settings**
   - Resolution: 1920x1080 at minimum
   - Frame rate: 30fps or 60fps
   - Cursor highlight enabled
   - Key press visualization (F4 press shown)

3. **Applications to Demonstrate**
   - Windows Terminal / PowerShell
   - VS Code (or similar code editor)
   - Chrome/Edge (browser)
   - Optional: Discord or game chat

4. **Audio Quality**
   - Clear voice-over (microphone in quiet room)
   - System audio for app sounds (if any)
   - Background music: subtle, technical/ambient

### Visual Style

- **Color Palette**: Match app branding (blue gradient, white)
- **Font**: Inter, Roboto, or similar clean sans-serif
- **Transitions**: Smooth fades, no harsh cuts
- **Overlays**: Semi-transparent dark overlays for text

### Graphics Needed

1. App logo (with glow effect for intro)
2. Feature cards with icons
3. Download button graphic
4. GitHub QR code (optional)
